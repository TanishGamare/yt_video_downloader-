# YouTube Video Downloader

A simple Python application that downloads YouTube videos using **yt-dlp** and lets you choose a download location through a Tkinter folder selection dialog.

## Features

* Download YouTube videos in the best available quality.
* Select the download folder using a graphical file dialog.
* Lightweight and easy to use.
* Built with Python, Tkinter, and yt-dlp.

## Requirements

* Python 3.8 or later
* `yt-dlp`

Install the required package:

```bash
pip install yt-dlp
```

## Usage

1. Run the program:

```bash
python YouTube.py
```

2. Enter the YouTube video URL when prompted.
3. Choose the folder where you want to save the video.
4. The download will begin automatically.
